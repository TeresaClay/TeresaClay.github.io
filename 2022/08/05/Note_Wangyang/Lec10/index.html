    Properties of discretisation schemes (蓝皮书 5.4)

    Numerical results will only be physically realistic when the disretisation scheme has certain fundamental properties

    Conservative

    • overall flux blance and conservation of $\phi$

    Boundedness

    • 源项的隐式处理

      • desirable feature is $a_P$ 与 $S_p$ 反号，帮助 $a_P$的对角占优

    • 对角占优

    • If the discretisation scheme does not satisfy the boundedness requirements it is possible that the solution does not converge at all, or contains wiggles

    Transportivness

    Accuracy

    • Taylor series

    CD (2nd order in uniform mesh)

    • unbounded

    Upwind (1st order)

    • false diffusion, smear

    More accurate higher-order schemes, which preserve upwinding for stability and sensitive to the flow direction.

    拓展

    QUICK，1992

    • 延迟修正 (deferred correction): 用上一步计算结果计算公式右侧第二项，这样就保证了系数项不会出现异号问题
    • 注意OpenFOAM使用的原生的QUICK

    小结

    • upwind scheme 无条件稳定，有界,没有 wiggles现象。但是会带来 false diffusion.
    • High order scheme (在计算流体指 2nd order) 当 $Pe$ 较大时，会带来 spurious oscillation or wiggles, 例如在计算 $k,\omega,\epsilon$等出现奇怪的非物理真实的负值，还会诱发计算的不稳定。
    • TVD 格式就是用来处理该问题: 有界，稳定
      • 找到精度高更稳定的格式
      • 构造保证单调性格式
      • total variation

    Total variation and TVD Schemes

    • 保证单调性TVD基本假设：

      1. No new local extrema must be created
      2. The value of an existing local minimum must be non-decreasing and that of a local maximum must be non-increasing

    • Total variation

    • 最初是随着时间变化的, 意味着 $TV(\phi^{n+1})\le TV(\phi^{n})$
